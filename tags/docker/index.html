Commandline Awesome: Dry

Een handige tool om te gebruiken terwijl je met Docker aan het werk bent is Dry. Dry is een grafische terminal app die het gebruik en beheer van Docker duidelijk weergeeft. Dry geeft realtime informatie over jouw draaiende containers en biedt eenvoudige mogelijkheden om bijvoorbeeld images en containers op te ruimen. Ook kunnen een stuk makkelijker container logs en statistieken worden ingezien. Kortom dry is een awesome tool als alternatief voor de Docker CLI die veel voorkomende commando’s wegneemt en zeker tijd gaat besparen. »
