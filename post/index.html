Splunk Live!

De dag begon met presentaties van bedrijven die uitleggen hoe ze Splunk gebruiken en wat hen dat oplevert. Vooral Coolblue en ABN Amro hadden hierin een mooi verhaal. CoolBlue legde uit hoe en waarom zij van een ELK-stack omgeving gemigreerd zijn naar AWS met Splunk, en ABN-AMRO liet zien hoe zij met Splunk hun SOC effectiever maken. »

Java One Update 2: De toekomst van Java

De chief architect of Java (Mark Reinhold) heeft tijdens de keynote de grootste veranderingen in Java 9 laten zien. Voor mensen die de ontwikkeling van Java een beetje bijhouden, weten dat de grootste wijziging project Jigsaw is. Het is nu mogelijk om je software in modules op te hakken, waarbij een module een groepering van packages is. Dit mechanisme kan je gebruiken naast het al bestaande (inferieure) classpath mechanisme. Tevens zijn alle bestaande java packages opgedeeld in 25 optionele en 1 verplichte module. »
